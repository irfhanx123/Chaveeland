# Chaveeland

CHAVEE - Exclusive Private Members Club & Curated Gatherings Website

## About

Chavee is an invite-only collective for visionary creators, intellectual leaders, and sensory pioneers. This repository contains the complete website for the Chavee private members club.

## Features

- **Home Page** - Hero section, philosophy pillars, interactive chamber teaser, testimonials carousel
- **Gatherings Page** - Dynamic events grid with search, filtering by category, and status badges
- **Application Portal** - Multi-step membership application form with validation
- **Thank You Page** - Digital nominee passport card with print functionality

## Tech Stack

- HTML5 (Semantic markup with SEO & Open Graph tags)
- CSS3 (Custom properties, glassmorphism, animations, responsive design)
- Vanilla JavaScript (No frameworks or dependencies)

## Design

- Dark luxury aesthetic with gold accent color palette
- Playfair Display + Outfit typography from Google Fonts
- Glassmorphism card components with ambient glow effects
- Fully responsive layout with mobile hamburger navigation

## Getting Started

Simply open `index.html` in your browser, or serve with any static file server.

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```
